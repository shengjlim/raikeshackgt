How to Use!

This application seperates audio files from YouTube into the three following modes:

  • Karaoke Track

  • A Cappella Track

  • Drum Track

Follow these steps to use:

  1. Find the URL of any Youtube Video.
  2. Select the mode you want
  3. Click Submit and wait for the file to generate
  4. Play the Youtube video
  5. Download the audio if you wish

We hope you have fun unmixing tracks!!!
